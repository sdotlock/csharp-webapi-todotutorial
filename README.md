# csharp-webapi-todotutorial

Following the ASP.Net Core documentation tutorial - [Create a web API with ASP.NET Core](https://docs.microsoft.com/en-gb/aspnet/core/tutorials/first-web-api?view=aspnetcore-5.0&tabs=visual-studio-mac)

## Required
* Visual Studio (Standard, Code or Mac)

* The latest .NET Core 5.x Target Framework

## Tutorial Creates The Following: 
* GET /api/TodoItems

* GET /api/TodoItems/{id}

* POST /api/TodoItems	

* PUT /api/TodoItems/{id}

* DELETE /api/TodoItems/{id}
